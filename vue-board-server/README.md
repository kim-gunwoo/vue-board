## server 기동 확인

```
package.json 파일 가장 먼저 확인해볼것
# main
시작 페이지 설정
# scripts
npm run 으로 시작하는 명령어 설정
# dependencies
배포시 의존하는 파일
# devDependencies
개발시에만 의존하는 파일

```

## Build Setup

```bash
# install dependencies
npm install

# serve start nodemon
npm run dev


```
